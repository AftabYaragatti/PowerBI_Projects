# 📌 Project Overview

The **Mobile Sales Dashboard** is an interactive Data Analytics project developed using **Python, Power BI, and Data Visualization libraries**.  
This project helps analyze mobile sales performance through interactive dashboards, graphical reports, and business insights.

The dashboard provides detailed analysis of:
- 📱 Brand-wise Mobile Sales
- 🌍 Region-wise Performance
- 💳 Payment Method Analysis
- 📈 Year-wise Sales Trends
- 👥 Customer Purchasing Insights

This project demonstrates practical implementation of:
✔ Data Cleaning  
✔ Data Analysis  
✔ Business Intelligence  
✔ Dashboard Reporting  
✔ Data Visualization
