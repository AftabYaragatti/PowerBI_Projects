# 📌 Project Overview

The **Mobile Sales Dashboard** is an interactive Data Analytics and Business Intelligence project developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Power BI.

This project transforms raw sales data into meaningful business insights through interactive dashboards and visual reports. It helps organizations analyze sales performance, customer purchasing behavior, revenue trends, payment methods, and regional performance to support data-driven decision-making.
